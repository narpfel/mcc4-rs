`mcc4-rs` -- Monte Carlo Connect Four in Rust
=============================================

Requirements
------------

* Rust (likely 1.8+, older stable might work) and `cargo`
* A terminal that understands ANSI escape sequences

Running
-------

```sh
cargo run --release --bin mcc4
```
